# Git이 뭔데
코드 형상관리, 버전관리

# git init
git 저장소 생성

# git clone
git 저장소 불러오기 

# git commit
변경 사항을 저장

# git push

## force push 금지!!

저장된 커밋 내용을 원격에 업데이트 

# git pull
원격에 있는 내용을 로컬에 불러오기

# branch 파기
가지치기, 여러 작업을 동시에 가능 

# merge가 머지
branch 간 합치는 작업 

# fork 하기
다른 사용자/organization의 repository를 내 계정에 복사

# PR 날리기
병합 요청 
- branch merge PR
- fork repository PR
